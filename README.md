# Органайзер для предметов пайки
Для поддержания порядка в рабочей зоне давайте создадим простой и удобный в использовании органайзер. В конкретном случае, это органайзер для предметов пайки.

# **:wrench: Создание настенного органайзера для хранения набора для пайки**

## **:mag: Обзор решения**
Универсальным и наиболее удобным решением для хранения большого количества инструментов является использование стенки типа skadis. Такая конструкция обеспечивает эргономичность, экономию пространства и удобство в использовании. Итак, наше изделие может быть выполнено при помощи разных технологий производства. Кроме того, оно должно встраиваться в концепцию данной стены и соответствовать главным критериям:
- мобильность;
- компактность;
- удобство использования;
- прочность и долговечность;
- экономность в использовании материала.

Необходимо сделать держатель для набора пайки в двойном экземпляре. Набор включает в себя: флюс, проволоку, очиститель жала, а также оловоотсос.

## **:hammer:Дизайн модели и производство**
### 1. Поиск аналогов

Существуют доски с углублениями для предметов пайки. Такой вариант очень удобен, его можно модернизировать и приспособить как настенный держатель.

### 2. Описание модели, выбор технологии производства.

Вместо углубления для катушки проволоки будем использовать болт, на который будет нанизана катушка своим отверстием вдоль главной оси. Таким образом удобно отрезать проволоку, не снимая катушку, или легко снять катушку, в зависимости от нужд человека. Чтобы конструкция не была слишком громоздкой и не создавала излишней нагрузки на крпеления, а это важно, ведь органайзер находится на стене, лучше разделить оловоотсос от остальных частей. Его можно спустить ниже, поместив на два крпеления. Что касается основной части держателя, она выполнена из фанеры, так как это наиболее быстрый и надёжный вариант. В данном случае будем использовать две фанеры, одна из них служит подставкой, а друая создает углубление, в которых могут держаться предметы пайки. Поэтому важно, чтобы верхняя фанера была толще, а именно 6мм, для нижней важно, чтобы она не утяжеляла конструкцию, поэтому её выполним из фанеры толщиной 3мм.

### 3. Проектирование модели.

Произведем измерения предметов набора пайки, сделав небольшие допуски, отразим их на чертежах составных частей модели. Затем, используя программу Компас 3D, создаем эскизы каждой из деталей и сделаем сборку.

### 3. Процесс создания

- крепления

Крепления произведены с помощью принтера аддитивной технологии Prusa 3D printer, получены слои и g-код для печати благодаря софту PrusaSlicer. Печать осуществляется пластиком PETG.
При подготовке детали к печати, а именно ее расположении на рабочем столике, необходимо учитывать нагрузку на крепления в соотвествии с расположением слоёв пластика.

<img width="300" height="400" alt="image" src="https://github.com/user-attachments/assets/c32fc625-06d5-4cfa-9ab0-bc9b0d4be4ff" />

- подставки

Для получения подставки необходим её чертеж в формате .dxf. Далее, нужно отредактировать чертеж в CorelDraw и задать нужные настройки для лазерной резки по фанере толщиной 3мм b 6мм.

<img width="1280" height="960" alt="image" src="https://github.com/user-attachments/assets/852061a0-e5f0-4143-8d3a-d5aa7f6525f8" />

### 4. Сборка

Будем использовать белую пропитку для фанеры, предварительно отшлифуя деталь от гари при вырезании лазером. Дадим высохнуть деталям и приступим к сборке. Для успешной сборки используйте 4 винта м3 для присоединения к креплениям, а также болт м4х70, который проходит через две фанеры и служит осью для проволоки. Как итог, мы получаем изделие, которое благополучно вписывается в рабочее пространство, долговечно, функционально и , как бонус, эстетично.

## **:key: Как использовать**

Для создания этого органайзера следуйте шагам:

1. Скачайте модель крепления, переведите в формат .stp или .stl, используя слайсер, отравьте на печать
2. Скачайте файл верхней подложки держателя и осуществите лазерную резку (расчёт на фанеру 6 мм)
3. Скачайте файл нижней подложки держателя и осуществите лазерную резку (расчёт на фанеру 3 мм)
4. Совершите сборку деталей с помощью винтов м3 и болта м4х70
